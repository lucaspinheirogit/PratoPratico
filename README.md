# Prato Prático

Aplicativo de receitas com busca rápida e avançada de pratos.

## Rodando o projeto

##### Baixe todas as dependências:

```
yarn 
```
ou
```
npm install
```
##### Execute o projeto em um emulador ou dispositivo físico:
```
react-native run-android
```
ou
``` 
react-native run-ios
```
